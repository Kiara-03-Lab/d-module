How to write ODEs

y' = 2y First derivative (y prime)
y'' + y = 0 Second derivative (harmonic oscillator)
y'' + 2y' + y = 0 Damped oscillator
y''' - y = 0 Third order equation

What is singularity analysis?

LCT (Log Canonical Threshold) Measures "how bad" the singularity is. Lower = worse.
Cusp (x¬≤ - y¬≥) Sharp point, LCT = 1/3
Node (x¬≤ - y¬≤) Crossing point, LCT = 1/2
Smooth (x¬≤ + y¬≤) No singularity at origin
